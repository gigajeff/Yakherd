# Origin

Yakherd emerged from the governance system used to manage a long-running,
multi-task photogrammetry engineering project. The reusable protocol was
separated from that product because a generic governance tool should not live
inside a domain repository.

The extracted V1 package contains no photogrammetry rules, datasets,
experiments, compute permissions, calibration assumptions, or project paths.
Historical Mosaic-origin package reviews are retained only as provenance for
the exact first-release bytes.

Yakherd is now independently owned, versioned, tested, and released from this
repository.
